    <h1>Saudações Personalizadas</h1>
    <p>Este é um simples script em Python que solicita ao usuário o nome e a idade, e então imprime uma saudação personalizada.</p>

    <h2>Descrição</h2>
    <p>Este script realiza as seguintes operações:</p>
    <ol>
        <li>Solicita ao usuário que digite seu nome e idade.</li>
        <li>Converte a idade fornecida para um número inteiro.</li>
        <li>Cria e exibe uma saudação personalizada usando o nome e a idade do usuário.</li>
    </ol>

    <h2>Como Usar</h2>
    <ol>
        <li>Clone este repositório para o seu ambiente local:
            <pre><code>git clone https://github.com/seu-usuario/seu-repositorio.git</code></pre>
        </li>
        <li>Navegue até o diretório do projeto:
            <pre><code>cd seu-repositorio</code></pre>
        </li>
        <li>Execute o script Python:
            <pre><code>python saudacao.py</code></pre>
        </li>
        <li>Siga as instruções no terminal para fornecer seu nome e idade.</li>
    </ol>

    <h2>Exemplo de Uso</h2>
    <pre><code>Digite o seu nome: João
Digite a sua idade: 25
Olá, João! Você possui 25 anos.</code></pre>

    <h2>Requisitos</h2>
    <p>- Python 3.x</p>
